# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in this project (MoonBase or MoonLight), please report it through [GitHub's private vulnerability reporting](https://github.com/MoonModules/MoonLight/security/advisories/new).

**Do not open a public issue for security vulnerabilities.**

## What to include

- Description of the vulnerability
- Steps to reproduce
- Affected components (CI/CD, firmware, frontend, dependencies, etc.)
- Impact assessment if possible

## Response

We will acknowledge reports as soon as possible and work on a fix before any public disclosure.
